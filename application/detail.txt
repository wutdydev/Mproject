cache : เป็นส่วนเก็บข้อมูลที่ประมวลผล
config : ตั้งค่าระบบต่างๆ ภายใน application
controllers : เราจะเขียนโค้ด PHP ควบคุมการทำงานของระบบ
core : " - -"
helpers : หากคุณต้องการเขียนฟังชั่นใช้เองในระบบสามารถสร้างและเก็บใว้ในที่นี่
hooks : ใช้สำหรับสร้างไฟล์ เพื่อเข้าไป เพิ่ม, แก้ใข Class ต่างๆ ที่ CI มีให้แล้ว
language : ที่เก็บไฟล์ภาษาต่างๆ ในกรณีที่ทำเว็บหลายภาษา
libraries : เป็นที่เก็บ Class ที่เขียนขึ้นเอง หากต้องการเรียกใช้ Class อื่นๆ ต้องเรียก get_instance ก่อน
logs : เป็นที่เก็บไฟล์บันทึกประวัติการทำงานภายใน application
models : คล้ายๆ libraries แต่ models จะสามารถเรียกใช้ libraries ที่คุณโหลดเข้ามาในระบบได้เลย
third_party : "- -"
views : ส่วนนี้เป็นการแสดงผล ส่วนใหญ่เราจะเขียนโค้ด html ที่นี่ แต่นามสกลุไฟล์ต้องเป็น .php เท่านั้น
